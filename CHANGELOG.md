# 更新日志 (Changelog)

所有重要的更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.3.2] - 2026-05-24

### 文档
- 更新 README.md 设置选项表格，添加「任务文件夹」设置项说明
- 简化 README.md 更新日志，引用 CHANGELOG.md
- data.json 添加 `customTaskFolder` 字段

## [1.3.1] - 2026-05-24

### 优化
- 文件夹选择器添加 30 秒缓存机制，提升大型库中打开设置面板的性能
- 创建任务前自动检查文件夹是否存在，如果被删除会自动重建
- 添加并发保护，防止重复创建任务文件

### 修复
- 修复 styles.css 版本号不一致的问题

### 文档
- 添加 LICENSE 文件（MIT 许可证）
- 添加 CHANGELOG.md 更新日志

## [1.3.0] - 2026-05-24

### 新增
- 「任务文件夹」设置项改为下拉菜单选择
- 自动扫描库中所有文件夹，提供便捷选择
- 支持自定义任务文件存储位置

### 优化
- 任务文件路径缓存机制，避免每次创建任务都遍历整个库
- 文件移动/删除时自动清除缓存

## [1.2.9] - 2026-05-24

### 新增
- 添加任务文件路径缓存机制
- 新增设置项「自定义任务文件夹」

### 优化
- 查找优先级：已存在的年度任务文件 > 自定义文件夹 > 默认文件夹
- 支持文件被移动后继续使用

## [1.2.8] - 2026-05-24

### 修复
- 修复文件移动后任务无法显示的问题
- 添加 `metadataCache.on("changed")` 事件监听，确保文件移动/重命名后元数据缓存更新完成再刷新视图

## [1.2.7] - 2026-05-24

### 修复
- 修复文件移动后任务无法显示的问题
- 添加 `metadataCache.on("changed")` 事件监听

## [1.2.6] - 2026-05-24

### 新增
- 月视图日历展示（6周网格布局）
- 农历显示（支持 1900-2100 年）
- 法定节假日标注（红色背景）
- 调休上班日标注（蓝色背景）
- 任务管理（创建、完成、删除）
- 多优先级标记（🔴 高 / 🟡 中 / 普通）
- 跨天任务支持
- 深色模式自动适配
- 移动端响应式设计

### 技术特性
- 内置 2022-2027 年节假日数据
- 支持从 timor.tech API 自动更新节假日
- 任务自动按月份分组存储
- 标准 Obsidian checklist 格式

## 版本说明

- **主版本号（Major）**：不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正
