### 核心功能
- **月视图日历** - 清晰的6周网格布局，查看每月任务一目了然
- **任务管理** - 快速创建、完成、删除任务
- **优先级标记** - 🔴 高优先级 / 🟡 中优先级 / 普通
- **跨天任务** - 支持多日任务横条展示
- **农历显示** - 自动显示中国传统农历日期

### 节假日
- **法定节假日标注** - 春节、国庆、五一劳动节等用红色背景高亮
- **调休日标注** - 周末上班日用蓝色背景标注
- **自动数据更新** - 启用插件时自动从 API 获取最新节假日数据

### 用户体验
- **深色模式** - 自动适配 Obsidian 主题
- **移动端适配** - 响应式设计，手机平板都能用
- **操作简便** - 点击日期即可添加任务，支持快捷键

## 使用方法

### 打开月历视图

1. 点击左侧边栏的日历图标
2. 或使用命令面板（`Ctrl/Cmd + P`）搜索「月历任务」
3. 选择「打开月历任务视图」

### 添加任务

1. 点击任意日期格子
2. 在弹窗中输入任务内容
3. 可选择：
   - 📅 **截止日期** - 默认当天
   - ⏰ **时间范围** - 非全天任务可设置起止时间
   - 🎨 **优先级** - 高/中/普通
   - 📆 **跨天任务** - 多日任务自动横条展示

### 任务格式

任务以标准 Obsidian checklist 格式存储：

```markdown
- [ ] 完成任务 📅 2026-04-22
- [x] 已完成任务 📅 2026-04-21
- [ ] 跨天任务 🛫 2026-04-22 📅 2026-04-25
- [ ] 带时间的任务 ⏰ 09:00~12:00 📅 2026-04-22
- [ ] 高优先级任务 🔴 📅 2026-04-22
- [ ] 中优先级任务 🟡 📅 2026-04-22
```

任务自动保存到 `任务/YYYY年任务列表.md` 文件中。

## 使用指南

### 打开月历视图

- 点击左侧边栏的 **日历图标**
- 或按 `Ctrl/Cmd + P` → 搜索「打开月历任务视图」

### 创建任务

#### 方式一：在笔记中手动编写（兼容 Tasks 插件格式）

```markdown
- [ ] 去超市买菜 📅 2026-04-22
- [x] 完成项目报告 🔴 📅 2026-04-20
- [ ] 团队会议 🟡 ⏰ 14:00 📅 2026-04-23
```

#### 方式二：在月历中点击日期直接创建

1. 在月历中点击任意日期格子
2. 输入任务内容
3. 选择优先级（高/中/普通）
4. 可选：指定时间 或 设为跨天任务
5. 点击「添加任务」

### 任务语法说明

| 标记 | 含义 | 示例 |
|------|------|------|
| `📅 YYYY-MM-DD` | 截止日期 | `📅 2026-04-22` |
| `⏰ HH:mm` | 具体时间 | `⏰ 09:30` |
| `🛫 YYYY-MM-DD` | 开始日期（跨天任务用） | `🛫 2026-04-20` |
| `⏫` 或 `🔴` | 高优先级 | `⏫ 重要任务` |
| `🔼` 或 `🟡` | 中优先级 | `🔼 一般任务` |

### 跨天任务

```markdown
- [ ] 公司年会 🛫 2026-05-01 📅 2026-05-03
```

跨天任务在月历中以**横条形式**显示，自动跨越多个日期格子。

### 设置选项

进入 **设置 → 社区插件 → 月历任务**：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 已完成任务显示方式 | 分段选择器：带删除线 / 正常显示 / 隐藏不显示 | 带删除线 |
| 显示农历 | 在日期下方显示农历和节气 | 开启 |
| 显示节假日 | 标注法定节假日和调休 | 开启 |
| 刷新节假日数据 | 从网络重新获取当年节假日信息 | — |
| 默认全天任务 | 新建任务默认不带具体时间 | 开启 |
| 每周第一天 | 日历每周起始日（周一 / 周日） | 周日 |
| 每日任务数量 | 每个日期格子最多显示的任务数 | 5个 |

## 项目结构

```
obsidian-monthly-tasks/
├── src/
│   ├── main.ts            # 插件主入口、设置面板
│   ├── MonthlyView.ts     # 月历视图组件 + 弹窗
│   ├── TaskModel.ts       # 任务数据模型与纯函数工具
│   ├── TaskParser.ts      # Vault 文件扫描与任务解析
│   ├── Calendar.ts        # 月历生成算法与日期工具
│   ├── LunarCalendar.ts   # 公历→农历转换（1900-2100）
│   └── HolidayManager.ts  # 法定节假日管理（timor.tech API）
├── styles.css             # 完整样式表（含深色模式和移动端响应式）
├── manifest.json          # Obsidian 插件元数据
├── package.json           # Node.js 项目配置
├── esbuild.config.mjs     # ESBuild 编译配置
├── tsconfig.json          # TypeScript 配置
├── .gitignore             # Git 忽略规则
└── README.md              # 本文件
```

## 构建与开发

```bash
# 安装依赖
npm install

# 开发模式（热重载）
npm run dev

# 生产构建
npm run build

# 构建产物
# main.js        → 编译后的插件入口
```
