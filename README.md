# 月历任务插件 (Monthly Tasks)

一款为 Obsidian 打造的滴答清单风格月视图任务管理插件，支持农历显示、节假日标注和调休信息展示。

---

## 功能特性

### 核心功能
- **月视图日历** - 清晰的6周网格布局，查看每月任务一目了然
- **任务管理** - 快速创建、完成、删除任务
- **优先级标记** - 🔴 高优先级 / 🟡 中优先级 / 普通
- **跨天任务** - 支持多日任务横条展示
- **农历显示** - 自动显示中国传统农历日期

### 节假日
- **法定节假日标注** - 春节、国庆、五一劳动节等用红色背景高亮
- **调休日标注** - 周末上班日用蓝色背景标注
- **自动数据更新** - 启用插件时自动从 API 获取最新节假日数据

### 用户体验
- **深色模式** - 自动适配 Obsidian 主题
- **移动端适配** - 响应式设计，手机平板都能用
- **操作简便** - 点击日期即可添加任务，支持快捷键

## 安装

### 方式一：BRAT（推荐）

1. 安装 [BRAT](https://obsidian.md/plugins?id=obsidian42-brat) 插件
2. 在 BRAT 设置中添加此插件仓库地址：
   ```
   https://github.com/your-username/obsidian-monthly-tasks
   ```
3. 启用插件即可

### 方式二：手动安装

1. 下载最新版本的 `monthly-tasks-1.3.2.zip`
2. 解压到 Obsidian 插件目录 `.obsidian/plugins/monthly-tasks/`
3. 重启 Obsidian 并启用插件

## 使用方法

### 打开月历视图

1. 点击左侧边栏的日历图标
2. 或使用命令面板（`Ctrl/Cmd + P`）搜索「月历任务」
3. 选择「打开月历任务视图」

### 添加任务

1. 点击任意日期格子
2. 在弹窗中输入任务内容
3. 可选择：
   - 📅 **截止日期** - 默认当天
   - ⏰ **时间范围** - 非全天任务可设置起止时间
   - 🎨 **优先级** - 高/中/普通
   - 📆 **跨天任务** - 多日任务自动横条展示
4. 按 **Enter** 键或点击确认按钮创建任务

### 任务格式

任务以标准 Obsidian checklist 格式存储：

```markdown
- [ ] 完成任务 📅 2026-04-22
- [x] 已完成任务 📅 2026-04-21
- [ ] 跨天任务 🛫 2026-04-22 📅 2026-04-25
- [ ] 带时间的任务 ⏰ 09:00~12:00 📅 2026-04-22
- [ ] 高优先级任务 🔴 📅 2026-04-22
- [ ] 中优先级任务 🟡 📅 2026-04-22
```

任务自动保存到 `任务/YYYY年任务列表.md` 文件中。

## 设置选项

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 显示已完成任务 | 在日历中显示已完成任务 | ✅ 开启 |
| 已完成隐藏删除线 | 打开后已完成任务将隐藏删除线 | ✅ 开启 |
| 默认全天任务 | 新建任务默认为全天 | ✅ 开启 |
| 每周第一天 | 周日或周一 | 周日 |
| 显示农历 | 显示中国传统农历 | ✅ 开启 |
| 显示节假日 | 显示法定节假日和调休 | ✅ 开启 |
| 每日任务上限 | 每天最多显示的任务数 | 5 |
| 任务文件夹 | 选择任务文件的存储位置 | 默认（任务） |
| 刷新节假日数据 | 从网络获取最新节假日 | 按钮 |

---

## 数据存储

插件数据存储在插件设置中：
- `data.json` - 用户设置和节假日缓存
- `任务/YYYY年任务列表.md` - 任务内容文件

---

## 节假日数据

### 内置数据
- 已内置 2022-2027 年节假日数据
- 无需网络即可显示

### API 数据源
- 使用 [timor.tech](https://timor.tech) API 获取最新数据
- 启用插件时自动刷新
- 可手动点击「刷新节假日数据」按钮更新

## 技术架构

### 模块结构

| 模块 | 说明 |
|------|------|
| TaskModel | 任务数据模型，包含任务解析、优先级提取等工具函数 |
| TaskParser | 任务解析器，负责解析 Obsidian 库中的任务，管理缓存 |
| Calendar | 日历工具，生成月历网格数据 |
| LunarCalendar | 农历转换，支持 1900-2100 年的农历日期转换 |
| HolidayManager | 节假日管理器，管理法定节假日和调休信息 |
| MonthlyView | 月历视图，渲染日历网格和任务列表 |
| CreateTaskModal | 创建任务弹窗，处理任务创建表单 |
| MonthlyTasksPlugin | 插件主类，管理插件生命周期 |
| MonthlyTasksSettingTab | 设置界面，提供用户配置选项 |

### 技术栈

- **语言**: TypeScript
- **框架**: Obsidian API
- **样式**: CSS（移动端优先设计）
- **数据**: 内置 + timor.tech API
